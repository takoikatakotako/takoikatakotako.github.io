概要

ZunTalkは、AI技術を活用した音声通話アプリケーションです。AIとの自然な対話を実現し、音声認識と音声合成技術を組み合わせたインタラクティブな通話体験を提供します。

主な機能

  • 発信音の再生
  • AIによるスクリプト生成
  • テキスト音声合成（TTS）
  • 音声認識と応答処理
  • 最大3分の通話時間管理

技術スタック

  • Swift - iOSアプリ開発（iOS 17.0以上、Xcode 15.0以上）
  • Go - バックエンドAPI（Go 1.21以上、Echoフレームワーク）
  • AI技術 - 自然言語処理と音声合成

プロジェクト構成

このリポジトリはモノレポ構成で、iOSアプリケーションとバックエンドAPIの両方を含んでいます。
