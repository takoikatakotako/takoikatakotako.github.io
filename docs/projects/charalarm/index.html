概要

Charalarmは、お気に入りの2次元キャラクターからの電話で目覚めることができるモバイルアプリです。好きなキャラクターを選んでアラームをセットすると、設定した時間にそのキャラクターから電話がかかってきて、楽しい朝の目覚めを提供します。

主な機能

  • キャラクター選択機能
  • アラーム設定・通知機能
  • キャラクターからの通話サービス
  • チャット・通話機能

技術スタック

  • Go - バックエンドAPI
  • Swift - iOSアプリ
  • HTML/CSS/JavaScript - フロントエンド
  • Terraform - インフラ構成管理

今後の予定

  • Android版アプリ対応
  • サーバーレス移行（Lambda/SNS/SQS など）
  • チャット機能の実装（DynamoDB 活用）
